# Агрегатор новостей

FastAPI-приложение для агрегации новостей с парсингом, категоризацией и персонализацией.

## Функциональность
- JWT аутентификация
- CRUD операции с новостями
- Парсинг новостей из различных источников
- Категоризация новостей
- Персонализированная лента новостей
- Веб-интерфейс с HTML страницами
